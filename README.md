# adbGUI
Desktop client for adb commands inspired from adb idea plugin for android studio 

### Features
1. Adb connect/disconnect
2. Adb screenshot
3. Adb screen record
4. Adb install/uninstall
5. Adb clear data
6. Adb reboot device
7. Adb push file
